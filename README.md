# ClauseCheck
Paste a lease, internship offer, or terms-of-service — get a plain-English breakdown of what you're actually agreeing to, with the risky parts flagged.

⚠️ Educational tool only — not legal advice.

## Stack
- Frontend: React + Vite + Tailwind v4
- Backend: Flask + OpenAI-compatible API

## Quick start
1. cd backend && pip install -r requirements.txt && cp .env.example .env && python app.py
2. cd frontend && npm install && npm run dev
